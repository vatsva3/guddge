# Deploy to Vercel

## Step 1: Install Vercel CLI
Open Command Prompt or Terminal and run:
```bash
npm install -g vercel
```

## Step 2: Deploy
1. Open Command Prompt
2. Navigate to your project folder:
```cmd
cd C:\Users\jyoth\Desktop\ADP
```

3. Run deployment:
```cmd
vercel
```

## Step 3: Follow the Prompts
- Email: Enter your email (or press Enter to link to GitHub)
- Log in to Vercel in your browser if prompted
- Which scope?: Select your account
- Link to existing project?: Press N (No)
- Project Name: Enter "guddge" or press Enter for default
- Directory?: Press Enter (uses current directory)
- Want to modify settings?: Press N (No)

## Step 4: Get Your Shareable URL
Vercel will show a URL like:
```
https://guddge.vercel.app
```

## Step 5: Share This URL
Copy the URL and share it with anyone!

---

## Alternative: Deploy via Vercel Website
1. Go to https://vercel.com
2. Sign up/Login
3. Click "Add New Project"
4. Choose "Upload" instead of GitHub
5. Drag and drop your ADP folder
6. Get instant shareable URL

---

## To Update Later
Just run `vercel` again in the folder to deploy changes.
